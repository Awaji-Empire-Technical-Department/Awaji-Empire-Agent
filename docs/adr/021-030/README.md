## 概要

- **ADR-021**: Discord 上でイベント参加フォームを完結させるシステムの設計（フォーム生成・応募受付・管理 UI）。
- **ADR-022**: イベント締切日時の自動処理スケジューラーと、部ごとの参加受付トグル UI の追加。
- **ADR-023**: Bot トークンを平文ファイル管理から `.env` + `python-dotenv` による環境変数管理へ移行し、セキュリティを強化。
- **ADR-024**: イベントフォームの回答フローのみギルド加入チェックを緩和（オフ会集計の取りこぼし防止）し、回答リンク共有ボタンを追加。DM は応募内容の控えとして位置づけ。
- **ADR-025**: スタッフ共同編集（ユーザー名検索でフォーム管理権限を共有）、回答の本人/管理者削除、オフ会当日の受付（チェックイン）モードを追加。
- **ADR-026**: ADR-024 の副作用で生じた、ギルド未加入者がフォーム回答後にダッシュボードへ流入できる不具合を修正。加入状態をセッションに保持し、ダッシュボード側でサーバーガード（403）＋回答完了画面のボタン無効化で対処。
