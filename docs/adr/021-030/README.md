# ADR 021–030: イベントシステム・インフラ整備

このディレクトリには、イベント参加フォームシステムの構築、運用自動化、インフラ整備に関する Architecture Decision Record が収録されています。

## 概要

- **ADR-021**: Discord 上でイベント参加フォームを完結させるシステムの設計（フォーム生成・応募受付・管理 UI）。
- **ADR-022**: イベント締切日時の自動処理スケジューラーと、部ごとの参加受付トグル UI の追加。
- **ADR-023**: Bot トークンを平文ファイル管理から `.env` + `python-dotenv` による環境変数管理へ移行し、セキュリティを強化。
