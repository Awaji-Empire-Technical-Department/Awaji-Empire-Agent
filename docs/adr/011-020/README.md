## 概要

- **ADR-011・014**: 配信コメントチャンネルの月次リセット機能と、配信中にリセットが走らないためのフェールセーフ機構。
- **ADR-012**: ゲーム種別を問わず利用できる汎用大会管理システムの設計方針。
- **ADR-013**: マリオカートワールドのラウンジシステム全体のアーキテクチャ（セッション管理・スコア集計・ロール付与）。
- **ADR-015**: 実績連動型の称号システムの設計（付与条件・ロール・UI）。
- **ADR-016〜020**: ラウンジシステムの継続的改善。リアルタイム化（WebSocket）、申告方式の変更、モバイル UI 対応、称号・モーダル表示バグの修正。
