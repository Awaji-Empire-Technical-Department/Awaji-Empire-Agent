# 🔔 通知マスミュート機能

## 1. 概要
サーバー内の特定チャンネルの通知設定を `@everyone` に対して一括上書きし、運用上発生する不要なプッシュ通知を抑制します。

## 2. 実行ロジック
複数のトリガーによって、常に定義通りの権限状態を維持します。

![Mass Mute Operation](./assets/mass-mute_operation.png)

## 3. 主な仕様
- **実行タイミング**: 
  1. Bot起動時（サービス復旧時）
  2. 定時タスク（日本時間 0:00, 8:00, 16:00）
  3. チャンネル作成時（動的な自動適用）
- **対象**: `config.py` で定義されたリストに完全一致するチャンネル名。
- **権限内容**: `mention_everyone=False`, `manage_webhooks=False` 等を強制適用。
