# Bot 権限要件：#配信コメント リセット機能

## 現状

`#配信コメント` の月次自動リセット機能を実装しました。
**現在、チャンネル再作成に Bot の管理者権限が必須です。**

---

## なぜ管理者権限が必要か

| 操作 | 権限 |
|------|------|
| チャンネル削除・再作成 | `manage_channels` 必須 |

- 通常のロール権限では Discord API が拒否
- テスト環境で管理者権限でのみ動作確認

---

## 実装の安全性

✅ 実行は厳格に制限
- VoiceKeeper 報告自動検知（月1日のみ）
- フォールバック cron（月2日 06:00 JST）
- スラッシュコマンド（管理者のみ実行可）

✅ 全操作を DB に記録・監査可能

✅ 破壊的操作は `#配信コメント` チャンネルのリセットのみ
