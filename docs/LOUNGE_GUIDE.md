# ラウンジシステム 運営ガイド

> 対象: ラウンジ運営に関わるスタッフ・ホスト  
> 最終更新: 2026-05-18

---

## 1. 概要：このシステムで何が変わるか

従来のMK8DX Loungeなどでは、募集・ルーム案内・スコア集計をすべてDiscordで行っていました。  
本システムでは **「ゲームのルーム管理はゲーム本体に任せ、それ以外（募集・スコア集計・ランキング）はWebアプリで行う」** という方針をとります。

| 作業 | 担当 |
|------|------|
| ルーム作成・CPU強さ・アイテム設定 | ゲーム本体（マリオカートワールド） |
| 募集告知・参加受付・スコア申告・ランキング表示 | **Webアプリ（本システム）** |
| ランク称号の付与 | **Webアプリ → Discordロール自動付与** |
| ボイスチャット（任意） | 参加者間で別サーバーを利用 |

---

## 2. ラウンジの流れ

### ホスト側

```
① ダッシュボードを開き、「マリオカートワールド ラウンジ」モードでセッションを作成する

② ゲーム本体でオンラインルームを立てる
   - CPU強さ: 強い
   - アイテム: ノーマル
   - インターバル: 10秒
   - 人数: 最大24人
   ※ これらの設定はゲーム側の責任。Webアプリからは操作できない

③ ゲーム画面に表示された「ルームID」をWebアプリのセッション画面に入力する
   → ダッシュボードに「ラウンジ募集中！」バナーが表示され、参加者に告知される

④ 参加者が集まったらレースを開始
   - 各レース終了後、コース名をWebアプリに入力してレースを登録
   - ※ コース重複はWebアプリが自動で検知して警告する

⑤ 各レースごとに参加者のスコア申告を承認する

⑥ 12レース終了後、セッションを終了する
   → 累計ランキングが確定し、ランク称号が自動付与される
```

### 参加者側

```
① ダッシュボードに「ラウンジ募集中！」バナーが出たら「参加する」ボタンを押す

② ホストから通知されたルームIDをゲームに入力して入室する
   ※ ルームIDはWebアプリのセッション画面で確認できる

③ レースが終わったら、ゲームの結果画面を見ながら「自分の順位」をWebアプリに入力する
   - 計算は不要。ゲームに表示された数字をそのまま入力するだけ

④ 回線落ちした場合は「回線落ち報告」ボタンを押す
   - CPUキャラクターの取得点数が自動で付与される
   - 同じレースで5人以上落ちた場合はそのレースが無効（ノーカウント）になる
```

## 3. ダッシュボードの表示イメージ

### 募集中バナー（セッション作成後〜終了まで表示）

```
┌──────────────────────────────────────────────────┐
│ 🏎️  ラウンジ募集中！                              │
│ ホスト: [ホスト名]  参加者: 8 / 24               │
│ モード: FFA  進行: 3 / 12 レース                  │
│                              [参加する]           │
└──────────────────────────────────────────────────┘
```

- セッションが存在しないとき、またはセッション終了後はバナーは非表示
- 「参加する」ボタンを押すと参加登録され、ルームIDが表示される

## 4. スコア申告とコース管理

### スコア申告

- 各レース終了後、参加者全員がゲームの結果画面に表示された「自分の順位」を入力する
- ホストがスタッフ操作パネルから「レース結果を承認」するとポイントが確定する
- 申告内容に矛盾があった場合は「⚠️ 矛盾あり」として表示され、ホストが直接修正できる

### コース重複ペナルティ

- 同一セッション内で同じコースが選ばれた場合、Webアプリが自動で警告を出す
- 「レインボーロード（単品）」と「ピーチスタジアム〜レインボーロード」は同一コースとして扱う

---

## 5. 称号とDiscordロール

ラウンジセッション終了後、参加者のMMR（レーティング）に応じた称号が自動付与されます。

| ランク | MMR帯 | Discordロール |
|------|------|------|
| 鉄 | 〜1999 | 自動作成・付与 |
| 銅 | 2000〜3999 | 自動作成・付与 |
| 銀 | 4000〜5999 | 自動作成・付与 |
| 金 | 6000〜7999 | 自動作成・付与 |
| プラチナ | 8000〜9999 | 自動作成・付与 |
| ダイヤ | 10000〜12999 | 自動作成・付与 |
| マスター | 13000〜 | 自動作成・付与 |

- ロールは **初めてそのランクに到達した参加者が出たタイミングで自動作成** される
- プレイヤーが装備する称号は1つだけ（Webアプリのダッシュボードで変更可能）
- Discord上に表示されるのは**装備中の称号ロール1つだけ**（YouTube Membershipなど既存ロールには影響しない）

---

## 6. チーム戦（2v / 3v）の場合

- セッション作成時にモードを「2v」または「3v」に設定する
- ホストがWebアプリ上でチームを編成し、各チームにタグを設定する
- 参加者はゲーム内でプレイヤー名の先頭にタグを付ける（例: `[ABC] プレイヤー名`）
- スコアは個人スコアの合算でチームスコアが計算される

## 7. Discordへの告知について（検討中）

現在、募集告知はWebアプリ上のバナーのみです。  
Discordの特定チャンネルへの自動投稿については、運営サーバーの方針を確認のうえ追加対応を検討します。

## 8. よくある質問

**Q. ゲーム内でルームIDはどこで確認できますか？**  
A. ゲーム本体のオンラインルーム作成後、ルーム待機画面に表示されます。

**Q. 回線落ちした場合のポイントはどうなりますか？**  
A. 落ちた際に使用していたCPUキャラクターの取得点数が付与されます。同じレースで同一キャラが複数落ちた場合は、報告が早かった順に高い点数が付与されます。

**Q. 申告を間違えた場合はどうなりますか？**  
A. ホストが承認前であれば差し戻しができます。承認後はホストが直接修正できます。

**Q. Webアプリにアクセスするにはどうすればいいですか？**  
A. Discordアカウントでログインします。淡路帝国サーバーに参加済みであることが条件です。
